邮件配置说明

此表单使用EmailJS服务发送邮件到 blackwhilecolor@Outlook.com。

要使其正常工作，您需要：

  1. 在 EmailJS官网 注册账号
  2. 创建一个电子邮件服务（如Gmail）
  3. 创建一个电子邮件模板
  4. 将下面的代码中的配置信息替换为您自己的：
// 在JavaScript部分替换这些配置
const EMAILJS_PUBLIC_KEY = '您的Public Key';
const EMAILJS_SERVICE_ID = '您的Service ID';
const EMAILJS_TEMPLATE_ID = '您的Template ID';

注意：此代码仅用于演示目的。在实际生产环境中，建议使用后端服务处理邮件发送。
